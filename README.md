# Flask Student Management CRUD API

This repository contains a simple RESTful Student Management API built using Python and the Flask framework. The project demonstrates the implementation of CRUD (Create, Read, Update, Delete) operations and serves as a beginner-friendly example for learning REST API development.

The API stores student information in an in-memory Python list and supports JSON-based request and response handling. It provides endpoints to create new student records, retrieve all students, update existing student details using both PUT and PATCH methods, and delete student records by their unique ID. The project also returns appropriate HTTP status codes and JSON responses to indicate the success or failure of each operation.

### Features

* Create a new student using the POST method.
* Retrieve all student records using the GET method.
* Update complete student information using the PUT method.
* Partially update student information using the PATCH method.
* Delete a student using the DELETE method.
* JSON request and response handling.
* Proper HTTP status codes (200, 201, 404).
* Simple and easy-to-understand project structure.
* Beginner-friendly implementation for learning Flask REST APIs.

### Technologies Used

* Python 3
* Flask
* JSON
* REST API

### Learning Objectives

This project helps developers understand Flask routing, request handling, JSON processing, RESTful API principles, HTTP methods, and CRUD operations. It is suitable for students, beginners, and anyone preparing for Python or Flask interviews.

This repository can be further enhanced by integrating a database such as MySQL, PostgreSQL, or MongoDB, implementing authentication with JWT, adding input validation, and deploying the application to cloud platforms such as Render, Railway, or Heroku.
